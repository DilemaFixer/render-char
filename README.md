# render-char

This project was created for educational purposes. The main goal was to understand and process letters, extract the glyph of a specific letter, and render it.

Building tutorial :

```
git clone https://github.com/DilemaFixer/render-char
cd render-char
git clone https://github.com/raysan5/raylib
gcc b.c -o b
./b
./build/main ./data/Alegreya/static/Alegreya-Black.ttf 'g'
```
